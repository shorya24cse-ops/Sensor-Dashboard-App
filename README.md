>> Sensor Dashboard App

This is a simple Android application developed using Java in Android Studio.
The app reads and displays real-time data from mobile sensors including Accelerometer, Light, and Proximity.

>> Features

* Displays Accelerometer data (X, Y, Z values)
* Shows Light sensor intensity (in lux)
* Detects Proximity (distance from object)
* Real-time sensor data updates
* Simple and user-friendly interface
